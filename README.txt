Fully functioning BinanceTradeBot that buys/sells cryptocurrency based on daily news sentiment from the crypto feeds.

### How It Works?

Pull and analyse the latest headlines from the followed crypto rss feeds. Provide an overview on the most mentioned coin. across all the headlines. Analyse the sentiment of each headline and categorise the output by coin. Place a buy-order if the compound sentiment is positive. The bot sells 95% of the coins it bought in order to avoid exceptions. "AssetBalance" dictionary stores the amount of coins the bot bought and currently holding.
